# Sarada Vilas College - Teacher Feedback Form
A simple and user-friendly **Teacher Feedback Form** for Sarada Vilas College that helps students submit feedback about teachers, courses, and subjects to improve teaching quality.

---

## Features
- Collects **Student Name**, **Course**, **Teacher**, and **Subject**.
- Dropdowns to rate **Teaching Quality**, **Communication Skills**, **Punctuality**, and **Overall Rating**.
- Radio buttons to select **Strength of the Teacher** (Friendly, Clear Explanation, Encouraging, Well Prepared).
- Includes **college logo** at the top and a **background image** for a clean, modern look.
- **Client-side validation** ensures all fields are filled before submission.
- Displays a **thank-you message** after submitting feedback.
- Option to **submit another response**.
